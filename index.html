Generative AI for accessibility

  1. Screen reader user installs a Chrome extension
  2. User encounters an inaccessible interactive element on a website
  3. User presses a keyboard shortcut
  4. Extension uses ChatGPT (or other LLM) API to generate accessible element
  5. Screen reader reads aloud accessible context
